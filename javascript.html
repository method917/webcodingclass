* Javascript 더 알아보기

* document객체
- 웹페이지 태그 삭제, 자식 태그 추가

* dom객체
- document의 심화

* window객체
- 웹브라우져 자체를 제어
- 주소 알기, 새창 열기, 화면 크기 알기 등

* ajax객체
- 웹페이지를 리로드하지 않고 정보를 변경
- 현대적 웹앱의 필수 테크닉

* cookie객체
- 웹페이지가 리로드되어도 현재 상태 유지
- 사용자를 위한 개인화된 서비스

* offline web application객체
- 인터넷이 끊겨도 동작하는 프로그램

* webRTC객체
- 화상통신

* speech객체
- 사용자 음성 인식, 음성 정보 전달

* webGL객체
- 3차원 그래픽 게임

* webVR객체
- 가상현실
